Project 1: Rule-Based AI Chatbot
This project is a simple chatbot developed using Python. The chatbot responds to predefined user inputs and demonstrates the basic working of a rule-based AI system.

Features

Handles unknown inputs with a default response
Includes an exit command to end the conversation
Uses Python dictionaries and loops for efficient response handling

User Input Handling
